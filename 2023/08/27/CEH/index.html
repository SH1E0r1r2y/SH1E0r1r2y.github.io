CEH v12 報考心得&想法分享

Created2023-08-27|Updated2023-09-27|writeup
securityCEHcertificateNotion

前言

學期初，一位熟識的學長告訴我，學校最近推動的深耕計畫有提供一門 CEH 公費培訓，我馬上決定抓住這個機會加入其中。很榮幸經過教授的推薦與教育部審核後，順利通過了相關流程。

CEH(Certified Ethical Hacker 道德駭客認證) 是屬於紅隊課程(藍隊 EC-Council 有另外一項證照 CND 可以報考)，對於有接觸過資安社課的人來說應該都有種熟悉的陌生人的感覺，而且仔細看蠻多都很像金盾獎的題目，個人是把它當作金盾獎的題庫來寫w((金盾獎範圍太大了不知道怎麼練習QQ

準備工作

查詢各種考試心得後，感覺刷考古題蠻重要的，不過講師也說，考古題至少兩成是錯的，因此我的方法是跟同學們一起寫共筆，共同討論進步。我還有使用 Notion 製作了一個個人版複習表，搭配遺忘曲線複習錯題，提供給本次深耕計畫 30 幾位參與人員分享 ((有時間會分享製作方式與連結

考試

  • Time：4 hours
  • 125題(題型：單選/多選)
  • 至少要寫一半時間再出來，證照比較不會被扣押w
  • 要先註冊考試系統
  • 考試卷的卷號(聽課聽完傳到 Gmail 上)：檢查三遍！如果考試不同
  • 線上比較高階的考試都需要護照！

教育訓練

  • 上課時數：40 hours
  • 本次獲得物資：
    • 遠端連線- 35天
    • iLab - 6個月
    • 兩次考試資格 - 1年內(因為是教育訓練所以到時候是共同考試)
    • 電子書 - ?((好像有人說期限是2025年
      • 教育訓練並沒有提供實體書，但有提供電子書的 token 在 aspen 平台，輸入 token 步驟可參考這篇文件：
  • 註冊操作
    這邊要特別注意 First name、Last name 是證照上面的名字，所以最好跟護照上面相同；Email Address 跟 Mobile Number 則是用於證照報名提醒、忘記密碼等用途
  • 工具下載：
    • 課程提供的遠端已經裝好所需工具了
    • My courses -> CEH -> 滑到最下面的 Downloads Tools
    • 離線使用電子書app(最多兩台裝置)：

後記

EC-Council證照效期為三年。 若要維持證照效期，三年內還需累計120點的ECE學分。
