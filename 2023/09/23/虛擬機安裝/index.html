Virtual Box 虛擬機

不要安裝Microsoft Store 的 Ubuntu，會變成wsl

一.Virtual Box 安裝

前往 Virtual Box 官網 https://www.virtualbox.org/

點擊 Download

根據電腦系統選擇相對應版本(下面以 Windows 為例)

二.下載 iso 檔案

需要有系統讓他在 Virtual Box 上執行，以 Ubuntu 22.04 穩定版為例(會下載較久，建議先安裝，或是在安裝時先進行下一步驟)

前往 Ubuntu 下載頁面 https://ubuntu.com/download/desktop

點擊 Download

三.虛擬機建立

至 Virtual Box 新增虛擬機


設定虛擬機名稱、位置
類型與版本請選擇 Linux 以及 Ubuntu (64-bit)


配置虛擬機記憶體大小，這裡設定 4GB (建議不要低於 2GB)


最後配置虛擬機硬碟大小，這裡選擇22.4版推薦的 25GB

完成後，點擊剛剛新增的虛擬機，再點擊右側上方的 設定，找到存放裝置

選擇CD圖案的按鈕，點擊，找到剛剛步驟二下載的iso檔案

四.開機與後續有利設定

第一次開機時要先設定，
如果登入遇到黑屏選擇try or install Ubuntu

其他只要跟著他的步驟一直 continue 就可以了(location 記得選台北)

自動調整視窗大小

如果想隨著視窗調整大小，需先掛載 Guest Additions CD 映像
開啟虛擬機後，在上方 裝置 選擇 插入 Guest Additions CD 映像 ，並選擇 run

重新啟動虛擬機就可以使用了

雙向剪貼簿

很多時候，我們會需要將本機的剪貼簿和虛擬機共用，例如下載或安裝時不想打長長一串URL

在虛擬機上方選擇 裝置 > 共用剪貼簿 > 雙向
如果效果沒有立即呈現，也可以重新啟動虛擬機看看喔!

補充.Kali設定檔

如果想使用 Kali 也可以(跟 Ubuntu 擇一即可)，
下面以Windows示範安裝流程

先把 Windows 保護關掉

❗❗ 可能會顯示偵測到病毒，因此辛辛苦苦等待下載完畢後又被瀏覽器刪掉，因此要先到設定 > 隱私權與安全性 > Windows 安全性


病毒與威脅防護 > 病毒與威脅防護設定 > 管理設定


把及時保護關掉，會出現一個”是否允許APP變更您的裝置”？ > 點擊是

❗❗❗下載完記得要趕緊開回來喔

下載流程

前往官網 https://www.kali.org/
點擊 Download


選擇 Virtual Machines (Recommended)> VirtualBox > 下載


下載後解壓縮(請勿使用Banditzip解壓縮，他也會辨識並將之刪除)，可以使用7zip
右鍵 > 顯示其他選項 > 7zip


看到下面這兩個就是成功下載


點擊「藍色 icon」的.vbox檔自動進入 VirtualBox
就可以使用了，它的預設帳號密碼為kali/kali
