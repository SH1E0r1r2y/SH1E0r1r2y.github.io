如何創建資料表？

SQL statement 型

  • 新增資料庫 > 給資料庫取名 > 按下確定
  • 在對應資料庫 > 按下[新增查詢] > 輸入指令

圖表GUI 型

  1. 新增資料庫 > 給資料庫取名 > 按下確定
  2. 點擊該資料庫旁邊的加號 > 右鍵點擊資料表 > 新增 > 資料表(T)
  3. 看到可輸入的畫面

成功！

sql 定義

row vs column

一「列」火車 -> 橫的 -> 資料庫中稱為"records"
兩「行」熱淚 -> 直的 -> column 是柱子的意思，所以是直的 -> 資料庫中稱為"attribute"

什麼是 private key? 什麼是 Foreign key?

tuple?relation?cardinality?Degree?
P.24頁

  • Tuple：就是 record，資料中的一列
  • Cardinality：record number，列的數量
  • Degree：column number，行的數量
  • Primary Key：用來識別每列資料的，每列 tuple 都僅有唯一常叫做id
  • Foreign Key：用來指向 Primary Key ，以關聯資料兩張 relation

What is relation?

relation：在 SQL 中，relation 是具有相同類型的數據列表，應該就是資料表

  • triple：三個 attribute的 relation
  • quadruple：四個…
  • sixtuple：六個…etc

What’s different between VARCHAR v.s. NVARCHAR?

  • Varchar：資料類型為 Non-unicode / 英文字元，僅支援ASCII values。最多可包含8000 char
  • Nvarchar：資料類型為 Unicode / 非英文字元，最多可包含4000 char
