架設伺服器

目標：架設CTF伺服器(像nc或ssh那種的連線)，如果可以的話平台更好

Q:如果虛擬機跟 docker 很像，那為什麼要在虛擬機上載 docker？
A:docker 是一種 container，而 container 跑的作業系統需要跟 host 相同，Linux Container 和 Windows Container 是無法共用的(當然也可以用wsl)，因此才會下載

Apache vs. Nginx:

  • Apache:最流行的Web伺服器軟體之一。它快速、可靠並且可通過簡單的API擴充
  • Nginx:是非同步框架的網頁伺服器，也可以用作反向代理、負載平衡器和HTTP快取
  • 待學:架設個人靜態網站：用 github 創一個 username.github.io 的 repo，再把網站 push 上去

快捷鍵

  • 一鍵修改變數名稱：「F2」
  • 快速把terminal叫出來:「Ctrl」+「alt」+「T」
  • 上、下引號：「Ctrl」+「[ or ]」+ 「下方建🔻」

虛擬環境virtual environment

如果有各種專案要執行，每個要執行的版本不同，為了避免版本打架，而獨立了各種資料夾，可依照需求安裝特定套件。

  • 啟動
    進入Scripts資料夾後，啟動activate.batmyenv01\Scripts\activate.bat
    在cmd看到前面有括一個(myevn01)就是成功了
  • 停用
    deactivate

pip vs.pipenv：

兩者皆用於管理虛擬環境中的套件
pip：使用requirements.txt

  • 如果專案的套件公司有要求，他會直接給你txt檔，可以使用pip install -r requirements.txt下載就行
  • 印出電腦目前的套件pip freeze > requirements.txt，用pipenv安裝的也會一併顯示

pipenv：即時的方式進行套件管理。它會自動生成一個 Pipfile，用於跟踪生產和dependencies.，不顯示用pip安裝的檔案
