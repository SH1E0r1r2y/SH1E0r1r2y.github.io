虛擬環境virtual environment

如果有各種專案要執行，每個要執行的版本不同，為了避免版本打架，而獨立了各種資料夾，可依照需求安裝特定套件。

  • 啟動
    進入Scripts資料夾後，啟動activate.batmyenv01\Scripts\activate.bat
    在cmd看到前面有括一個(myevn01)就是成功了
  • 停用
    deactivate

pip vs.pipenv：

兩者皆用於管理虛擬環境中的套件
pip：使用requirements.txt

  • 如果專案的套件公司有要求，他會直接給你txt檔，可以使用pip install -r requirements.txt下載就行
  • 印出電腦目前的套件pip freeze > requirements.txt，用pipenv安裝的也會一併顯示

pipenv：即時的方式進行套件管理。它會自動生成一個 Pipfile，用於跟踪生產和dependencies.，不顯示用pip安裝的檔案
